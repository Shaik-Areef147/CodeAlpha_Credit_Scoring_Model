# Credit Scoring Model

## Overview

The Credit Scoring Model is a Machine Learning web application that predicts an individual's creditworthiness based on financial and personal attributes. The project uses a Random Forest Classifier to analyze input data and classify credit risk.

## Features

* Data preprocessing and cleaning
* Feature encoding and transformation
* Random Forest Classification Model
* Credit Risk Prediction
* Flask-based Web Interface
* Model Deployment on Local Host

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Flask
* Joblib
* HTML

## Installation

1. Clone the repository

git clone <repository_link>

2. Navigate to project folder

cd Credit_Scoring_Model

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py

5. Open browser and visit

http://127.0.0.1:5000
